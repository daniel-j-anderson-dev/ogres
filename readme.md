OGRES: Other Global Regular Expression Search
learning project that will mimic GREP

<b>Usage:</b> ogres [OPTIONS] [PATTERN] [FILE]

# Examples:

#### Usage:
```sh
    ogres "unix" "./new.txt"
```

#### Output:
```sh
    Found "unix" in "fully/qualified/path/new.txt" @ line 3, word 4
```
